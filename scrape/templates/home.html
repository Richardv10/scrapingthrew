Instructions

Welcome to Rich's scraper tool (Alpha build v1.2)
This is provided "as is" and is in active development, feel free to fork the project for personal use.
It's built in django to make use of python modules. Providing basic DOM extraction via Beautiful Soup, and Selenium integration for dynamic content scraping.
Built in awareness of robots.txt, and limits recursive scraping depth to avoid overloading servers. Use responsibly!
I am integrating Ai features, and improving the UX, check the Github Readme for Dev diary updates NOTE: SELENIUM IS CURRENTLY NOT FUNCTIONAL ON HEROKU ECO DYNOS, SO IF THE FEATURE IS WORKING IT'S PROBABLY BECAUSE I HAVE AN INTERVIEW COMING UP
